## Getting Started

### Installation

```
git clone https://github.com/PDUBB3/gym-guru-client.git
cd dish-up
npm i
```

You will also need to add your .env following the variables in the .env.EXAMPLE file.

### Set up the database

In MySQL workbench:

```
DROP DATABASE IF EXISTS dishup_db;

CREATE DATABASE _db;
```

To seed the data:

```
npm run seed
```

### Launch the app

```
npm run start
```
